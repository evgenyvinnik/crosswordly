# Internationalization (i18n)

This project uses [i18next](https://www.i18next.com/) and [react-i18next](https://react.i18next.com/) for internationalization.

## Current Languages

- English (en) - Default
- Spanish (es)

## Structure

### Configuration

The i18n configuration is located in `src/i18n/config.ts`. This file initializes i18next with all available languages and sets up the default language.

### Translation Files

Translation files are JSON files located in `src/i18n/locales/`:

- `en.json` - English translations
- `es.json` - Spanish translations

### Translation Keys

The translation files are organized into the following namespaces:

- `settings.*` - Settings menu strings
- `stats.*` - Statistics dialog strings
- `levelSelect.*` - Level selection screen strings
- `shelves.*` - Level category/shelf labels
- `levels.*` - Level titles and descriptions

### Usage in Components

To use translations in a React component:

```tsx
import { useTranslation } from 'react-i18next';

function MyComponent() {
  const { t } = useTranslation();
  
  return <h1>{t('settings.title')}</h1>;
}
```

### Level Localization

Level titles and descriptions are localized using the `useLocalizedLevels` hook from `src/levelLocalization.ts`:

```tsx
import { useLocalizedLevels } from './levelLocalization';

const localizedLevels = useLocalizedLevels(levels);
```

### Language Switcher

The `LanguageSwitcher` component is available in the Settings menu, allowing users to change the language at runtime.

## Adding a New Language

1. Create a new JSON file in `src/i18n/locales/` (e.g., `fr.json`)
2. Copy the structure from `en.json` and translate all strings
3. Import and add the translations in `src/i18n/config.ts`:
   ```ts
   import frTranslations from './locales/fr.json';
   
   i18n.use(initReactI18next).init({
     resources: {
       // ... existing languages
       fr: {
         translation: frTranslations,
       },
     },
     // ... rest of config
   });
   ```
4. Add the language to the `LanguageSwitcher` component in `src/components/LanguageSwitcher.tsx`:
   ```tsx
   const LANGUAGES = [
     // ... existing languages
     { code: 'fr', name: 'Fran√ßais' },
   ];
   ```

## What is NOT Translated

The following items are intentionally not translated:

- Word puzzles (words from `words.ts`)
- The "Crosswordly" brand name
